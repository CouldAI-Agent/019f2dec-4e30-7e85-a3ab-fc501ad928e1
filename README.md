# Telegram Anime Bot

Bu Telegram bot orqali siz foydalanuvchilarga kino va animelarni kod orqali yuborishingiz mumkin. Shuningdek, majburiy obuna va premium tizimi mavjud.

## Xususiyatlar
- Majburiy obuna
- Premium foydalanuvchilar
- Kod orqali anime qidirish
- Admin panel

## O'rnatish
1. `.env.example` faylini `.env` deb o'zgartiring va o'z ma'lumotlaringizni kiriting.
2. Kerakli kutubxonalarni o'rnating: `pip install -r requirements.txt`
3. Botni ishga tushiring: `python bot/main.py`
